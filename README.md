# Baraka Biz - Wholesale Trading & E-Commerce

A professional static home page for Baraka Biz, a multi-category trading enterprise engaged in wholesale, import, export, and e-commerce activities.

## Overview

Baraka Biz operates as a diversified trading company specializing in:

- **Food & Beverages**: Wholesale trading of packaged food items and beverages
- **Frozen Products**: Wholesale trading of frozen foodstuff with proper compliance
- **Mobile Electronics**: Trading of mobile phones and related accessories
- **Import & Export**: International trade operations across approved product categories
- **E-Commerce**: Online retail sales through digital platforms
- **Distribution**: Wholesale supply to retailers, resellers, and institutional buyers

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the website using the menu

### Option 2: Deploy Online

## Customization

### Update Company Information
Edit `index.html` to modify:
- Company name and logo
- Contact information
- Product descriptions
- Service offerings

### Customize Colors
Edit `styles.css` to change the color scheme:
```css
--primary-color: #2c3e50;      /* Main brand color */
--secondary-color: #3498db;    /* Accent color */
--accent-color: #e74c3c;       /* CTA button color */
```

### Add Functionality
Enhance `script.js` to add:
- Contact form submission
- Product filtering
- Customer testimonials
- Newsletter signup
- Analytics integration
